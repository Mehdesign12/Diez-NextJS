# Diez Agency - Landing Page Reproduction

Ce projet est une reproduction haute-fidélité de la landing page de "Diez Agency", conçue avec une approche **Mobile First** radicale.

## 🌟 Expérience Mobile ("App-Like")

L'expérience sur mobile a été priorisée pour offrir une sensation d'application native :

*   **Navigation Immersive** : Menu overlay plein écran avec animations d'entrée séquentielles et effet glassmorphism.
*   **Smart Swiping** : Les sections longues (Portfolio, Témoignages, Prix) se transforment en **carrousels horizontaux (Snap Scroll)** sur mobile pour éviter le défilement vertical excessif.
*   **Touch Feedback** : Réactivité immédiate des éléments interactifs au toucher.
*   **Interface Épurée** : Navigation flottante (Pill) toujours accessible sans encombrer l'écran.

## 🎨 Design & Animations

*   **Design System** : Tailwind CSS avec une palette de couleurs personnalisée (`diez-orange`, `diez-dark`, `diez-base`).
*   **Animations Desktop** :
    *   Hero : Effet "Word Rise" et "Gradient Shimmer".
    *   Scroll : Apparitions progressives et effets de parallaxe.
    *   Curtain Reveal : Pied de page qui se dévoile en fin de scroll.
    *   Text Highlight : Surlignage contextuel des mots-clés.
*   **Internationalisation** : Site bilingue (FR/EN) avec détection instantanée et traduction complète sans rechargement.

## 🛠 Stack Technique

*   **HTML5** : Sémantique.
*   **Tailwind CSS** : Styling utilitaire.
*   **Vanilla JS** : Performance maximale, zéro dépendance lourde.

## 📂 Structure

*   `index.html` : Code source unique (Structure + Style + Logique).
*   `images/` : Assets graphiques.

## 🚀 État du Projet

*   ✅ **Intégration Pixel-Perfect** : Reproduction fidèle du design original.
*   ✅ **100% Responsive** : Du mobile (320px) aux écrans larges (4k).
*   ✅ **Interactions Avancées** : Toutes les animations sont fonctionnelles.
*   ✅ **Prêt pour la Production** : Code propre et documenté.

## 🔗 Déploiement

Utilisez l'onglet **Publish** pour mettre le site en ligne.
Pour éditer le contenu, modifiez l'objet `translations` dans le bas du fichier `index.html`.
